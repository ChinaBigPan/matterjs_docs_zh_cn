# SAT

[英文原地址](https://brm.io/matter-js/docs/classes/SAT.html)

`Matter.SAT`模块包含了使用分离轴理论(Separating Axis Theorem)检测碰撞的方法。

## 方法

### Matter.SAT.collides(bodyA, bodyB, previousCollision) → Collision

使用分离轴定理检测两个物体之间的碰撞。

**参数：**    
`bodyA` Body  
`bodyB` Body    
`previousCollision` Collision

**返回值：**   
Collision 碰撞
