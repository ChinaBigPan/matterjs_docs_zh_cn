# Axes

[英文原地址](https://brm.io/matter-js/docs/classes/Axes.html)

`Matter.Axes`模块包含了用于创建和操作一组轴的方法。

## 方法

### Matter.Axes.fromVertices(vertices) → Axes

从给定的顶点创建一组新的轴。

**参数：**    
`vertices` Vertices   

**返回值：**

Axes 从给定的顶点创建的一组新的轴。

### Matter.Axes.rotate(axes, angle)   

以给定的角度旋转一组轴。

**参数：**    
`axes` Axes  
`angle` Number
